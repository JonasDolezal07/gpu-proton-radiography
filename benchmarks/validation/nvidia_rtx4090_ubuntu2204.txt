Environment setup (RunPod / headless Docker)
--------------------------------------------
NVIDIA's libGLX_nvidia.so.0 requires an X11 connection to initialise
even for compute-only Vulkan. On headless servers, use Xvfb:

  apt-get install -y xvfb
  Xvfb :99 -screen 0 800x600x16 &
  export DISPLAY=:99
  export VK_ICD_FILENAMES=/etc/vulkan/icd.d/nvidia_icd.json

These two env vars must be set before running validate.py or any
proton_tracer invocation. prad's batch compute path does not create
a window or surface; Xvfb merely satisfies the GLX init requirement.
