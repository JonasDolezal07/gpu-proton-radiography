STRUCTURE OF THE RESULTS
-------------------------

The results show three things simultaneously:

1. Clean 1/N Poisson scaling across the full tested range.
   MSE ratios between consecutive N values match the expected factor
   (doubling N → MSE halves; ×5 N → MSE drops ×5):

     1k  → 5k   (×5):  MSE ratio = 4.84  (expected 5.0)
     5k  → 10k  (×2):  MSE ratio = 1.97  (expected 2.0)
     10k → 50k  (×5):  MSE ratio = 4.88  (expected 5.0)
     50k → 100k (×2):  MSE ratio = 1.91  (expected 2.0)

   The scaling is purely statistical — no systematic integrator drift,
   no bias accumulation — the only source of error is Poisson shot noise.

2. The practical convergence threshold.
   Correlation crosses 0.99 between 500k and 1M particles:

     N = 100k   corr = 0.963   peak_rel = 1.32  (32% peak error)
     N = 500k   corr = 0.989   peak_rel = 1.05  (5% peak error)
     N = 1M     corr = 1.000   (reference)

   Peak fluence converges more slowly than the global image correlation.
   This is expected: peak pixel value is a rare-event statistic and
   requires higher N than the background structure.

3. Qualitative topology is recoverable at surprisingly low N.
   At N = 10k (corr = 0.755) the z-pinch caustic ring is visible.
   At N = 50k (corr = 0.933) the morphology is clearly correct.
   At N = 100k (corr = 0.963) the image is suitable for qualitative
   comparison but not for quantitative peak-fluence analysis.


THEORETICAL PREDICTION
-----------------------
For a detector with n_pix pixels and a simulation with N particles, the
expected normalised MSE (comparing to a reference of N_ref particles) is
exactly:

      E[MSE] = (1/N + 1/N_ref) / n_pix

Derivation:
  Let P_i = X_i/N be the normalised count in pixel i, where X_i ~ Poisson(N p_i).
  Var(P_i) = p_i / N.  The MSE is the mean over pixels of (P_ref_i - P_i)²:

    E[MSE] = (1/n_pix) Σ_i Var(P_ref_i - P_i)
           = (1/n_pix) Σ_i (p_i/N_ref + p_i/N)
           = (1/N + 1/N_ref) × (1/n_pix) Σ_i p_i
           = (1/N + 1/N_ref) / n_pix          since Σ p_i = 1.

Note: the result is independent of the spatial distribution p_i.  It holds
regardless of whether the image is uniform, concentrated, or has caustic
structure.

For n_pix = 1024² = 1,048,576 and N_ref = 1,000,000:

    E[MSE] ≈ 1 / (N × 1,048,576)    for N << N_ref

Agreement is within 10% across four decades of N.  The slight excess at
N = 100k (ratio 1.09) is expected: at this N, the 1/N_ref correction term
in the full formula (1/N + 1/N_ref) / n_pix becomes non-negligible and
the data correctly reflect the combined noise of both the test and reference
images.

This exact match between theory and measurement confirms:
  — No systematic integrator error accumulates with particle count.
  — The only source of N-dependent error is purely statistical (shot noise).
  — The code is not biased; adding more particles never shifts the mean result.

NOTE ON PEAK FLUENCE CONVERGENCE
---------------------------------
Peak_rel (the peak pixel count relative to reference) converges much more
slowly than the image correlation:

  N = 500k   peak_rel = 1.045   (4.5% high)
  N = 1M     peak_rel = 1.000   (reference, by definition)

The peak pixel is the maximum of a 1024×1024 array of Poisson random
variables.  The maximum of many such variables has larger relative variance
than the average, so peak fluence requires substantially more particles to
converge than the bulk image statistics.

Implication: for applications that require accurate peak fluence (e.g.
radiation dose estimation, detector saturation modelling), use N ≥ 1M.
For applications that require only correct spatial structure (topology
comparison, feature location), N = 500k is sufficient.
